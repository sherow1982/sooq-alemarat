# سوق الإمارات - دليل الإعداد

## تثبيت TinaCMS

تم إعداد TinaCMS بنجاح على موقعكم! الآن يمكنكم تحرير المحتوى بصريًا.

## الملفات المضافة:

- `package.json` - حزم Node.js المطلوبة
- `tina/config.ts` - ملف إعدادات TinaCMS
- `config/site.json` - إعدادات الموقع
- `pages/home.md` - الصفحة الرئيسية

## خطوات التشغيل:

1. **تثبيت الحزم:**
   ```bash
   npm install
   ```

2. **تشغيل TinaCMS:**
   ```bash
   npm run dev
   ```

3. **الوصول لواجهة التحرير:**
   افتح المتصفح واذهب إلى: `http://localhost:4001/admin`

## ميزات التحرير المتاحة:

### 1. تحرير المنتجات
- إدارة قائمة المنتجات بالكامل
- إضافة وتعديل وحذف المنتجات
- رفع الصور مباشرة
- تحرير الأوصاف بمحرر نص غني
- إدارة الأسعار والخصومات
- تصنيف المنتجات

### 2. تحرير الصفحات
- تحرير المحتوى بواجهة بصرية
- إضافة صفحات جديدة
- تحرير النصوص والعناوين

### 3. إعدادات الموقع
- تغيير اسم الموقع ووصفه
- إدارة معلومات الاتصال
- روابط وسائل التواصل الاجتماعي

## واجهة عربية بالكامل

جميع عناصر واجهة التحرير تم ترجمتها للعربية لسهولة الاستخدام.

## الأمان والحماية

سيتطلب الأمر إعداد متغيرات البيئة للحماية:
- `NEXT_PUBLIC_TINA_CLIENT_ID`
- `TINA_TOKEN`

يمكن الحصول عليها من موقع tina.io بعد إنشاء حساب مجاني.
